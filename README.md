# sandbox_for_csharp
* C#用サンドボックス
* 機能のサンプルコードとかを置いている

## コーディングルール
* サンプルを書く機能の単位でEx〇〇クラスをExBaseを継承して追加する。
* Exec() の定義内に実行するサンプルコードを記述する。
